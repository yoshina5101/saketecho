## サービス概要
その日の気分から最適なお酒（日本酒・焼酎・ビール・ウィスキー）を提案するサービスです。
また飲んだお酒を記録し、自分の嗜好を可視化する「sake手帳」がコアとする機能です。
このアプリでは自身の好みの発見と、新しいお酒との出会いに寄与します。

## このサービスへの思い・作りたい理由
私は普段お酒を飲む際、「今日はどんなお酒を飲みたいのか？」が自分でもよく分からず、あまりその時の気分にあっていないお酒を選んでしまうことが多くありました。
また、飲んだお酒の記録を紙やメモアプリで残していた時期もありますが、「どの味が好きで、どれが苦手なのか」等をあとから振り返りにくいという課題も感じていました。
そこで、「気分でお酒を提案してくれて、好みの傾向もわかるアプリ」 があれば便利だと考え、本サービスの開発に至りました。日常の小さな選択を楽にして、新しい出会いを楽しむ、そんな体験を作りたいという思いがあります。

## ユーザー層について
① お酒は好きだけれども、どれを選べばよいか迷う人
→ 気分に合わせて提案されることで、選択の負担が減る。

② 新しい銘柄やジャンルを開拓したい人
→ 自分の飲酒傾向が可視化され、新しい発見につながる。

③ 日本酒やウイスキーに興味はあるが詳しくない初心者
→ 気分ベースで気軽に選べ、専門知識を必要としない。

## サービスの利用イメージ
①今日の気分を選ぶ（例：スッキリ・しっとり・じっくり飲みたい）。
②気分に合ったおすすめの酒が表示される。
③気になるものをタップして味わい・度数・産地などの詳細を確認。
④実際に飲んだら「sake手帳」に飲酒記録として登録。
⑤記録が蓄積されると、自分の好みの傾向やよく飲むジャンルが見えてくる。
→ 「今日の気分に合ったお酒は？」「自分はどんな味が好き？」がすぐ分かる体験を提供できると考えます。

## ユーザーの獲得について
SNS（X・Instagram）で「気分で選ぶお酒診断」としてシェアすることを可能にする。
お酒レビューや食事ペアリング記事など、検索の流入を意識したコンテンツを用意する。
初心者でも理解しやすい味の分類ガイドなどを提供し、検索価値を高める。

## サービスの差別化ポイント・推しポイント
● 他サービスとの違い
一般的なレビューサイトは「知識がある人向け」であると感じることが多い。
しかし本アプリは その日の気分ベースでレコメンドする点で独自の体験価値を提供することが可能であると考えています。

● 推しポイント
飲酒記録が蓄積されることで、
　■ 自分の嗜好が分かる
　■ 気分別のおすすめ精度が上がる
　■自分専用のsake手帳が育っていく感覚がある

## 機能候補
【MVP】
①ユーザー登録（Devise）
②気分選択（Mood）
③おすすめ酒のレコメンド表示
④お酒一覧 / 詳細（Beverage）
⑤飲酒記録（TastingLog）

【本リリース】
①お気に入り登録
②記録から好みの傾向を可視化（レーダーチャート）
③SNSでの共有機能
④食事とのペアリング提案
⑤お酒のレビュー機能
